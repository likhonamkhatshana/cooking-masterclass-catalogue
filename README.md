# Cooking Masterclass Catalogue

## Project Overview
This is a Vue 3 single-page application that displays cooking courses offered by expert chefs. Users can browse available courses and save them to a wishlist.

## Features
- Dynamic course catalogue
- Wishlist counter
- Sold Out status display
- Responsive layout
- Vue components
- Props and Events

## Technologies Used
- Vue 3
- Vite
- JavaScript
- CSS

## Installation

1. Clone the repository
```bash
git clone <repository-url>
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```
